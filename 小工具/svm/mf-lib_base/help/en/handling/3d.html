3D Visualizations

Overview for key commands

move view:
1. Click on an area in the view and drag the mouse with the right button pressed.
or
2. Navigate by using the arrow keys.

Zoom in and zoom out:
Navigate by using the zoom buttons.

Rotate view:
1. Click on an area in the view and drag the mouse with the left button pressed.
or
2. Navigate by using the rotaion buttons.
